# Finland live train map

A Hack Club Fallout project.

A map with LED's marking the route for intercity trains across Finland.

# Repository structure / table of contents

**Each directory contains its own README file with more information on the specific section.**

## /pcb

- Contains KiCad design files
- More about PCB design in `/pcb/DESIGN.md`

### KiCanvas viewer
View interactive PCB design in [KiCanvas](https://kicanvas.org/?github=https%3A%2F%2Fgithub.com%2FHekinav%2FFinland-Live-Train-Map%2Ftree%2Fmain%2Fpcb)

**NOTICE**: KiCanvas does not properly render everything, plase hide layers User.1-User.5 to view power circuitry

## /design

- Contains Map Design ideas and plans as Inkscape SVGs
